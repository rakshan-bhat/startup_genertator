# Startup Generator
A Flask-based web application designed to generate innovative startup ideas using Groq's LLM API.

## Overview
This project leverages the power of large language models (LLMs) to create a web application that generates startup ideas based on user input.

## Features
- **User Input:** Users can input a prompt or theme for the startup idea.
- **LLM Integration:** Groq's LLM API is used to generate startup ideas based on the user's input.
- **Web Interface:** A simple and intuitive web interface built with Flask allows users to interact with the application easily.

## Requirements
- **Python 3.9+**
- **Flask**
- **Groq LLM API Key**
- **python-dotenv** for environment variable management
